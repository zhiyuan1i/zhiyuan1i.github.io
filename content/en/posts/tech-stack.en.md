## Tech Stack

This site is built with the following technologies:

| Technology | Purpose |
|------------|---------|
| [Hugo](https://gohugo.io/) | Fast static site generator written in Go |
| [PaperMod](https://github.com/adityatelange/hugo-PaperMod) | Clean and elegant Hugo theme |
| [GitHub Pages](https://pages.github.com/) | Free static site hosting |
| [GitHub Actions](https://github.com/features/actions) | Automated deployment |

## Features

- **Blazing Fast**: Hugo's Go implementation ensures sub-second builds
- **SEO Friendly**: Built-in Open Graph, Twitter Cards, and structured data
- **Dark/Light Mode**: Automatically follows system theme
- **Full-text Search**: Site-wide search powered by Fuse.js
- **Responsive Design**: Perfectly adapted for mobile devices
- **Multi-language**: Supports both Chinese and English
